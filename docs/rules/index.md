# SeewoGeek社群规则


:::tip 伙伴们，欢迎你来~
为什么要建立这样一个社群？
一个人的力量是有限的，如果面对问题感到无助，来群里和大家沟通探讨，小伙伴们的善意和自己的主动都可以让自己变得更好。
 
放心，社群里并没有必须完成的任务，也不一定非要发言。你可以只是围观，看到更多人的困惑是如何被回应的。
 
为了更好地从社群中获益，你可以阅读和收藏这份社群规则介绍，了解社群打开方式。我们群里见！
:::
## 社群主要聊什么？

- 生活日常，平时和朋友/网友怎么聊天，在这里就怎么聊啦~
- 生活中的疑问和困难，包括但不限于技术问题

群里不建议深度处理个案困扰，这需要专业知识和足够的context，个别极端案例可能属于专业心理咨询的工作范围，无法通过群聊进行。

## 社群禁止聊什么？

在大多数时候，群内交流是轻松和友好的。  
但为了保护所有成员的体验，也为了避免不必要的风险，以下内容被视为**社群底线**，请务必注意：
:::warning 群聊底线
- 群内禁止发布或讨论任何违反我国法律法规的内容，包括但不限于：软色情、擦边内容，以及 **V*N** 相关内容。
    
- 涉及性取向、性别议题的内容，请避免展开具有争议性、对立性或带有挑衅意味的讨论。

- 群内禁止发布广告、助力、推广、人头车等类似宣传

- 发言内容在短时间内引发多位群友明确表达不适、反感或投诉，经管理员评估认为已明显影响群内交流体验的。
    
- 若因违规行为导致社群被封禁或解散，群主不承担任何责任。请大家共同维护群内环境。
:::

:::warning **注意事项**

为了保证您和其他群友的聊天质量，**我群对群成员聊天内容有相应的基本限制**，包括但不限于

- 若长期发布明显与事实严重不符、且在提醒后仍拒绝修正的技术性言论，并对群内讨论质量造成持续影响的，管理员可能会介入处理。
- 在某些不合时宜的聊天语境中，不要发送诸如“6” 、“。。。”之类的可能会在当前语境引人烦躁且毫无意义的词句。
- 在某些不合时宜的聊天语境中，不要发送与群聊主题不符的内容。
- 在某些不合时宜的聊天语境中，不要发送诸如“有你们这些人才，我就放心躺平了”、“群里全是大佬，就我啥也不会”之类的可能会在当前语境包含“阴阳怪气”“绿茶”等意思的词句。

**(注，若在实际事件处理中，违规人员对“不合时宜的聊天语境”的定义有异议的，由“希沃售后业绩冲击部管理组”进行最终裁定)**

:::

### 管理处置说明

在出现上述情况时，群主或管理员可能会采取以下措施：

- 打断当前话题，并提醒相关成员注意群规；
- 视情况发布临时中止群内讨论的通知。
    

如在提醒后，相关行为仍未及时停止，群主或群管理员有权采取 **「移除社群」** 的强制措施。

被移除社群后，我们有权不再通过您的加群申请（不论是否使用小号）。

具体采取何种措施，将由管理员根据事件对群内讨论质量与成员体验的影响程度综合判断。

### 社群里怎么提问和分享？

为了尽可能从提问和答疑中获益，会建议大家将提问本身也当作一个澄清困扰的实践——理解和描述自己的困扰，同样也是解决问题很重要的一步。

这也更有利于大家提供更清晰准确的支持，提问时可以参考以下方式帮助澄清：

:::tip 正确的提问方式

- 你想做什么？
- 你遇到什么问题？
- 你做过什么？
- 你尝试过什么解决方案？
- 结果是什么？

**PS: 提供越多有效信息越能得到帮助**
:::

:::warning 错误的提问例子

- 有人在吗？
- 有人会吗？
- 有大佬在吗？
- 有人懂吗？
- 我要刷机！
- 我要什么什么的。

**PS: 大家都忙，请尊重彼此时间**
:::

## 关于自行拉群或添加群友好友

- 不可在社群里直接发其它群的二维码，如果第一次出现会提醒，提醒之后，继续的话，不排除会移除出群；
- 如果与相熟或互相信任的群友已经添加好友的，互相希望单独组群，原则是「自发进行、自负其责」。
- 对于发起好友申请的小伙伴，除非经过对方的允许或邀约，不建议在无沟通前情的情况下主动添加对方好友。
- 对于收到好友申请邀请的小伙伴，如果与申请方之前无沟通、或不知对方添加缘由、或不认可对方，建议不通过好友申请。
- 即根据自己的主观意愿，在知情、自主愿意的前提下，再考虑通过好友申请。

群管理员有权将任何做出明显不合适行为或影响其他群友利益的人移除出群，移除出群不影响后续练习继续进行。

另外，**群主和群管理员一般不会主动添加群U个人好友。** 若您收到类似邀请，请您注意核实对方身份信息，以免遭遇电信诈骗。

群规的最终解释权归 SeewoGeek 管理组所有。
我们也会根据社群实际情况持续修订规则，希望在清晰边界与良好体验之间保持平衡。
